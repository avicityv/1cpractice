Основные объекты:

Документы

Отчеты

Формы

2. Справочники
Создадим справочники для хранения основных данных:

Справочник "Категории товаров"

Поля: Наименование, Описание.

Пример: Верхняя одежда, Обувь, Аксессуары.

Справочник "Товары"

Поля: Наименование, Категория, Артикул, Цена, Количество на складе, Описание.

Пример: Куртка зимняя, Кеды, Шарф.

Справочник "Поставщики"

Поля: Наименование, Контактная информация.

Пример: ООО "Ткани и шерсть", ИП Иванов.

Справочник "Клиенты"

Поля: ФИО, Телефон, Email, Адрес доставки.

Пример: Петров Иван, +79991234567, petrov@mail.ru.

3. Документы
Создадим документы для управления процессами:

Документ "Приходная накладная"

Поля: Номер, Дата, Поставщик, Товары (табличная часть с полями: Товар, Количество, Цена).

Пример: Приход товара от поставщика.

Документ "Заказ"

Поля: Номер, Дата, Клиент, Товары (табличная часть с полями: Товар, Количество, Цена).

Пример: Заказ от клиента.

Документ "Реализация товара"

Поля: Номер, Дата, Заказ, Товары (табличная часть с полями: Товар, Количество, Цена).

Пример: Отгрузка товара клиенту.

4. Регистры
Создадим регистры для учета:

Регистр "Остатки товаров"

Измерения: Товар.

Ресурсы: Количество, Сумма.

Пример: Учет остатков на складе.

Регистр "Продажи"

Измерения: Товар, Клиент.

Ресурсы: Количество, Сумма.

Пример: Учет продаж.

5. Отчеты
Создадим простые отчеты:

Отчет "Остатки товаров на складе"

Поля: Товар, Количество, Сумма.

Пример: Список товаров с остатками.

Отчет "Продажи за период"

Поля: Товар, Количество, Сумма, Клиент.

Пример: Отчет по продажам за месяц.

6. Формы
Создадим простые формы для справочников и документов:

Форма списка товаров

Фильтр по категории, поиск по наименованию.

Форма документа "Заказ"

Табличная часть с товарами, кнопка "Рассчитать сумму".

Форма отчета "Остатки товаров"

Фильтр по категории.
